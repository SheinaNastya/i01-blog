			  Справочник для дизайнера тем

1. Файлы

common.html      -- общий шаблон для всех страниц

Эти шаблоны вставляются на место {CONTENT} в common.html:
article.html     -- шаблон для статей
blog.html        -- шаблон для веблога
gallery.html     -- шаблон для галереи
stats.html       -- шаблон для статистики

Это различные меню:
menu.html        -- боковые меню
sitemap.html     -- раскрытое меню (как в списке статей)

2. {Плейсхолдеры} и <!-- BEGIN блоки --><!-- END блоки -->

2.1 Common.Html

{TITLE}          -- заголовок страницы
{CONTENT}        -- содержимое страницы
block            -- блок
small_block      -- уменьшенный блок
errors           -- ошибки и варнинги при генерации страницы

Также здесь доступны шаблоны статистики (см. разд. Stats.Html).

2.2 Article.Html

rubrika          -- рубрика/подзаголовок статьи
pic              -- картинки, приложенные к статье
{TITLE}          -- заголовок статьи
copyright, date  -- автор/источник статьи, дата написания
annotation       -- аннотация/предисловие к статье
{TEXT}           -- текст статьи
file             -- файлы, приложенные к статье

2.3 Blog.Html

mu_*             -- см. разд. Menu.Html
post_message     -- содержимое выводится при просмотре списка постингов
post_comment     -- содержимое выводится при просмотре комментариев
uplevel          -- содержимое выводится вместе с формой нового поста
messages,
  message        -- сообщения
comments,
  comment        -- комментарии
{LEVEL}          -- уровень комментария
postform         -- форма нового поста
pager            -- выбор страницы

2.4 Gallery.Html

{TITLE}          -- название галереи
annotation       -- аннотация/предисловие
pic              -- картинки

2.5 Stats.Html

{OLDEST}         -- начало отсчеты статистики
{UNIQUE_TODAY}   -- уникальных за прошедшие 24 часа
{UNIQUE_MONTH}   -- уникальных за месяц
{UNIQUE_ALL}     -- уникальных всего
{BOTS_TODAY}     -- роботов за прошедшие 24 часа
{BOTS_MONTH}     -- роботов за месяц
{BOTS_ALL}       -- роботов всего
request          -- запросы поисковых машин
referer          -- откуда приходят на сайт

2.6 Menu.Html

См. документацию на компонент HTML_Menu (http://pear.php.net).

2.7 Sitemap.Html

См. документацию на компонент HTML_Menu (http://pear.php.net).
